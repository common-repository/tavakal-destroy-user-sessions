Free light plugin to kick out user after being afk, or inactive for long time even if the browser is closed. You can change the time and users roles anytime in option page.

== Description ==

By default in wordpress sessions have expiration date, it might be too long for some kind of websites. With this plugin, you can set N time for afk (away from keyboard) users, before ending their session. For example if N time is 1 hour, users will be logged out after 1 hour from the last activity time.


> #### Tavakal - Destroy user sessions Features
>
> - **Set maximum inactive time before ending user session**.<br />
> - **Set roles that should apply this logic.**<br />

== Installation ==


= Installing the plugin - Simple =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Tavakal - Destroy user sessions** and click "*Install now*"
2. Activate the plugin
3. Go to General tab under WordPress Settings menu.
5. Find the "Tavakal - Destroy user sessions" option and save the options.
